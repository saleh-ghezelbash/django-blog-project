# Django Blog Project

A complete blog platform built with Django featuring posts, tags, comments, categories, and user management.

## Features

- **User Management**: Custom user model with profiles, authentication, and authorization
- **Posts**: Rich text editor (CKEditor), featured images, tags, categories
- **Comments**: Nested comments with moderation
- **Categories & Tags**: Organize content with categories and tags
- **Search**: Full-text search functionality
- **Responsive Design**: Mobile-friendly Bootstrap 5 templates
- **Admin Panel**: Full-featured Django admin interface
